# REST full Application

Example social media application

1. Retrieve all users 	- GET 		/users/
2. Create a users		- POST 		/users/
3. Get specific user	- GET		/users/{id} -> Example like /users/1
4. Delete specific user	- DELETE	/users/{id} -> Example like /users/1

5. Retrieve the all posts for specific user 	- GET 	/users/{id}/posts
6. Create a post for a user						- POST  /users/{id}/posts
7. Retrieve the details of post					- GET  	/users/{id}/posts/{post_id}
